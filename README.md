# Redes
Minhas anotações de estudo sobre protocolos de rede, conceitos e diagramas. Uso essas anotações pros meus projetos pessoais e pra desafios práticos como CTFs e no OverTheWire.

## O que tem aqui
- **01 - Conceitos de Rede por distância:** classificação de redes por alcance (PAN, LAN, MAN, WAN)
- **02 - TCP e IP, DNS, DHCP, NAT:** protocolos fundamentais da pilha TCP/IP
- **03 - Estados TCP:** os estados de uma conexão TCP, do handshake ao encerramento
- **Diagramas e Imagens:** diagramas de apoio pras anotações acima
